# Boarscape

Small Three.js third‑person playground using the included `boar3.glb` as the player.

## Run

```bash
npm install
npm run dev
```

Then open the URL Vite prints (usually `http://localhost:5173`).

## Controls

- **WASD**: move
- **Shift**: run
- **Space**: jump
- **Mouse drag**: rotate camera
- **Mouse wheel**: zoom
